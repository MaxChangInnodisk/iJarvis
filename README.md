# iJarvis
iJarvis is an demonstration with `STT`, `VLM`, `LLM` tech on `Jetson` devices.

| Author | Email | Additional
| ---    | --- | ---

## How to use?
```bash
# Start all service
python3 tools/launcher.py start

# Stop all service
python3 tools/launcher.py stop
```
